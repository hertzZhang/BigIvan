### `tab | table`

```markdown
| Names | Age | Cool |
| ----- | :-: | ---: |
| He    | 18  |  Yes |
```


### `del | delete`

```markdown

```

### `li | list`

```markdown
1. First ordered list item
2. Another item
3. And another item
```

### `uli | ulist`

```markdown
- Unordered list can use asterisks

* Or minuses

- Or pluses
```

### `a | link`

```markdown
[link](http://zhanghe.cool 'Link's title')
```

### `al | alink`

```markdown
<me.example@example.com>
```

### `img | image`

```markdown
![alt text](http://Image's src 'Image's title')
```

### `imgwh | imagewh`

```markdown
![alt text](Image's src){:height='36px' width='36px'}
```
